# kontraksi---Bunda-Helper---Contraction-Timer
aplikasi web sederhana dan responsif yang dirancang untuk membantu para ibu hamil (Bunda) dalam menghitung durasi serta jarak interval kontraksi menjelang persalinan. Aplikasi ini dilengkapi dengan algoritma rekomendasi otomatis untuk memberikan sinyal kapan waktu yang tepat untuk pergi ke rumah sakit atau menghubungi tenaga medis.
